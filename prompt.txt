# Prompt — Créer le site officiel de BloomUI

Crée un site web **100 % original** présentant **BloomUI**, un nouveau langage de design pour les interfaces web.

Le site **doit lui-même être conçu selon les principes de BloomUI**. Il ne doit pas seulement parler de BloomUI : il doit en être la démonstration.

## Identité

Nom : **BloomUI**

Slogan : **Interfaces that Bloom. 🌸**

Mission :
Créer des interfaces modernes, artistiques et vivantes qui restent intuitives, performantes et professionnelles.

---

# Direction artistique

Le site doit donner l'impression d'un produit premium, innovant et créatif.

Le visiteur doit avoir envie d'explorer chaque section.

Inspirations :

* Apple
* Figma
* Linear
* Framer
* Raycast
* Stripe

Mais avec une personnalité beaucoup plus expressive et chaleureuse.

---

# Le style BloomUI

Le site doit respecter les principes suivants :

• Animations fluides et naturelles.
• Beaucoup de micro-interactions.
• Dégradés élégants.
• Éléments organiques.
• Formes flottantes.
• Cartes modernes.
• Légers effets de verre (Glassmorphism).
• Ombres douces.
• Bordures lumineuses.
• Profondeur.
• Arrière-plan vivant avec particules discrètes.
• Légère parallaxe.
• Icônes modernes.
• Illustrations originales.
• Responsive.
• Accessibilité.
• Excellentes performances.

Aucune animation ne doit être inutile : chaque effet doit améliorer l'expérience.

# Règles importantes

Le site doit être spectaculaire sans être surchargé.

Le design doit sembler vivant.

Les animations doivent être douces.

Chaque section doit surprendre.

Le visiteur doit immédiatement comprendre la philosophie de BloomUI simplement en naviguant sur le site.

Le site doit donner l'impression que BloomUI est un véritable langage de design reconnu, avec une identité forte, une documentation en préparation et une vision à long terme.

L'objectif n'est pas de créer un simple portfolio, mais une expérience immersive qui montre ce que signifie réellement : **Interfaces that Bloom.** 🌸
